# 嵌入式开发工程师能力模型

## 核心技能要求
### 基础硬技能
- C/C++ 编程：指针、内存管理、数据结构、算法
- 微控制器：STM32、ESP32、ARM Cortex-M 系列
- 实时操作系统：FreeRTOS、RT-Thread、uCOS
- 通信协议：UART、I2C、SPI、CAN、USB
- 硬件基础：原理图阅读、示波器使用、GPIO/ADC/PWM/Timer

### 进阶技能
- Linux 驱动开发：字符设备、平台设备驱动、设备树
- 网络协议栈：TCP/IP、LwIP、MQTT
- 嵌入式 AI：TinyML、边缘计算
- 安全开发：安全启动、加密存储、固件安全

## 薪资参考范围（2024年）
- 应届生：8K-15K
- 1-3年经验：12K-25K
- 3-5年经验：20K-40K
- 5年以上：35K-60K+

## 热门行业方向
1. 物联网（IoT）
2. 汽车电子（ADAS、车载ECU）
3. 消费电子（智能家居、可穿戴）
4. 工业控制（PLC、工业网关）
5. 医疗电子（医疗设备嵌入式系统）
