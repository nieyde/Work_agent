# JobPlanning Agent

---

基于 LangGraph ReAct 范式二次开发智能体编排内核，自维护 9 个工具链的自主决策调度，实现简历解析→人岗匹配→差距诊断→规划生成→记忆闭环的全链路 Agent 自动编排

实现基于 PydanticOutputParser + FAISS 向量检索引擎的结构化知识架构，支持 PDF / Word / TXT 3 种简历格式零标注解析，覆盖简历写作/面试方法/岗位模型 3 大专业知识库的 RAG 增强问答

设计 JSON 本地的长期记忆持久化方案，解析后自动触发画像落盘，跨 Session 档案秒级还原，二次访问零摩擦续接历史对话上下文

搭建基于 Streamlit 的对话交互前端，集成简历上传、多轮对话、一键规划、历史档案管理等 5+ 功能模块，一条命令本地启动完整 Agent 应用
